# Operation-Courgette

Irrigation system using an ESP32, water reservoir, water level sensor and a water pump to water courgette plants.

This project has been inspired by an irrigation project by Giovanni Aggiustatutto I came across on [Instructables](https://www.instructables.com/ESP32-Irrigation-System-With-Water-Reservoir/).  I have used this project as a basis for my hardware choices; water pump, water level sensor and wiring.

Tasks:
- Turn water pump on/off.
- Water level sensor.
- Web server for manual control and feedback.
- Bluetooth to wake up wi-fi web server to save power usage.
- Solar battery power, this is a long term goal after I have a working solution.

Hardware:
- ESP32 (exact model to be decided)
- Water pump
- Water level sensor
- Other stuff to be added when I start the project

The code will be written using VSCode and the PlatformIO plug-in.
